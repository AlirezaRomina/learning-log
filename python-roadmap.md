# Python Learning Roadmap

A structured path to learn and master Python for data science and analytics.

---

## Phase 1 — Python Fundamentals (Weeks 1–3)

**Goal:** Get comfortable with Python syntax and core concepts.

- [ ] Variables, data types (int, float, str, bool)
- [ ] Operators and expressions
- [ ] String formatting (`f-strings`, `.format()`)
- [ ] Input/output (`print`, `input`)
- [ ] Conditionals (`if`, `elif`, `else`)
- [ ] Loops (`for`, `while`, `break`, `continue`)
- [ ] Functions (defining, parameters, return values, scope)
- [ ] Lists, tuples, sets, dictionaries
- [ ] List comprehensions
- [ ] Basic error handling (`try`, `except`)

**Resources:**
- [Python.org Official Tutorial](https://docs.python.org/3/tutorial/)
- *Automate the Boring Stuff with Python* (free online)

---

## Phase 2 — Intermediate Python (Weeks 4–6)

**Goal:** Write clean, reusable, and well-structured Python code.

- [ ] File I/O (reading/writing `.txt`, `.csv` files)
- [ ] Modules and packages (`import`, `from`)
- [ ] Object-Oriented Programming (classes, inheritance, `__init__`, `self`)
- [ ] Decorators and closures
- [ ] Iterators and generators
- [ ] Lambda functions and higher-order functions (`map`, `filter`, `zip`)
- [ ] Virtual environments (`venv`) and package management (`pip`)
- [ ] Working with the standard library (`os`, `sys`, `datetime`, `collections`)
- [ ] Writing and running tests (`unittest` or `pytest`)

**Project idea:** Build a CLI tool to log and summarize your learning sessions.

---

## Phase 3 — Data Handling & Analysis (Weeks 7–10)

**Goal:** Use Python to work with real data.

- [ ] NumPy — arrays, vectorized operations, broadcasting
- [ ] Pandas — Series, DataFrames, reading CSVs, filtering, groupby, merging
- [ ] Data cleaning (handling nulls, type casting, duplicates)
- [ ] Exploratory Data Analysis (EDA) workflow
- [ ] Matplotlib — line charts, bar charts, scatter plots, histograms
- [ ] Seaborn — statistical visualizations
- [ ] Jupyter Notebooks for interactive analysis

**Project idea:** Analyze a real dataset (e.g. from Kaggle or data.gov) and write up findings.

---

## Phase 4 — SQL + Python Integration (Weeks 11–12)

**Goal:** Connect SQL knowledge to Python workflows.

- [ ] `sqlite3` — run SQL queries from Python
- [ ] `SQLAlchemy` — ORM basics and database connections
- [ ] `pandas.read_sql()` — load query results into DataFrames
- [ ] Writing reusable data pipeline scripts

**Project idea:** Pull data from a SQLite database, transform it in Pandas, and export a report.

---

## Phase 5 — Advanced Python & Best Practices (Weeks 13–16)

**Goal:** Write production-quality Python code.

- [ ] Type hints and `mypy`
- [ ] Logging (`logging` module)
- [ ] Context managers (`with` statement, `__enter__`/`__exit__`)
- [ ] Concurrency basics (`threading`, `asyncio`)
- [ ] Regular expressions (`re`)
- [ ] Code style and linting (`black`, `flake8`, `ruff`)
- [ ] Packaging a Python project (`pyproject.toml`, `setup.py`)
- [ ] Working with APIs (`requests`, JSON parsing)
- [ ] Environment variables and secrets management

---

## Phase 6 — Data Science & Machine Learning (Weeks 17–24)

**Goal:** Apply Python to machine learning and data science problems.

- [ ] Scikit-learn — train/test split, preprocessing, model fitting, evaluation
- [ ] Supervised learning (linear regression, decision trees, random forests)
- [ ] Unsupervised learning (k-means clustering, PCA)
- [ ] Model evaluation metrics (accuracy, precision, recall, RMSE)
- [ ] Feature engineering
- [ ] Pipelines in scikit-learn
- [ ] Introduction to deep learning with PyTorch or TensorFlow (optional)

**Project idea:** Build and evaluate a predictive model on a real dataset.
